Java Installation - 

1. Download and Install Java:-
Visit the website: https://www.oracle.com/java/technologies/downloads/#jdk19-windows
Download the installer and run

2. Set the Environment Variables
	
	a. Path:-
	- Copy path from windows explorere 	
	- Open settings and set System variable - Path - Add/Edit
	- C:\Program Files\Java\jdk-18.0.2.1\bin;


	b. JAVA_HOME :-
	- Create a System Variable - JAVA_HOME
	- Copy the path - C:\Program Files\Java\jdk-18.0.2.1;
	- It is required in spring boot applications

3. Check Java Version
Open CMD

cmd-> java -version
